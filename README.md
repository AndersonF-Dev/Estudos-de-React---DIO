## 🚀 Miniguia de Estudo (Entrega Final)

### 📝 Resumo: Componentes e Hooks
Componentes em React são blocos de construção de interface reutilizáveis. Com a introdução dos **Hooks**, gerenciamos estado e ciclo de vida diretamente em funções, tornando o código mais legível do que a antiga estrutura de classes.

### 📖 Glossário React
* **JSX:** Sintaxe que estende o JavaScript, permitindo escrever estruturas que parecem HTML dentro do código JS.
* **Virtual DOM:** Uma cópia leve do DOM real que o React usa para comparar mudanças e atualizar a tela de forma muito mais rápida.
* **Props:** Propriedades passadas de um componente pai para um componente filho (são imutáveis).
